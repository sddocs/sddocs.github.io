# Документация для Shopify Custom Thank you Page #

### Requirements ###

> Ruby 2.5.0 и выше;
> RubyGems;
> GCC и Make.

#### Как запустить ####

```
gem install jekyll bundler
bundle exec jekyll build
bundle exec jekyll serve
```
